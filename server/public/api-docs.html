Colors API

Base URL: http://localhost:3001/api

GET/colors

Get all colors

curl http://localhost:3001/api/colors

POST/colors

Create new color (hex field required)

curl -X POST http://localhost:3001/api/colors \ -H "Content-Type: application/json" \ -d '{"name": "Blue", "hex": "#0000FF"}'

PUT/colors/:id

Update existing color

curl -X PUT http://localhost:3001/api/colors/1 \ -H "Content-Type: application/json" \ -d '{"name": "Updated Blue"}'

DELETE/colors/:id

Delete color

curl -X DELETE http://localhost:3001/api/colors/1
