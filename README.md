# T3 Demo

A minimum setup for React + TypeScript + Vite + Supabase app

## Quick Start

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Setup environment:**

   ```bash
   cp .env.example .env
   ```

   Then edit `.env` with your Supabase credentials from [supabase.com/dashboard](https://supabase.com/dashboard)

3. **Start development:**
   ```bash
   npm run dev
   ```

## Project Structure

- `/src` - Frontend React app
- `/server` - Public API
